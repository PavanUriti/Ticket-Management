Welcome to the Bus Ticket Management System

This platform is implemented using Node.js, Express.js, and MongoDB. It allows administrators to manage user registration and bus registration. Additionally, it provides APIs for admin registration, customer registration, login, and ticket booking.

Key Features

  • Admin Registration
  • Customer Registration
  • Login
  • Tickets Booking
  • Bus Management

How to Use

To get started, navigate to the provided API endpoints for registration, login, and ticket booking. Use the provided APIs to manage buses efficiently.
